# 输出

- `%+v` 打印包括字段在内的实例的完整信息
- `%#v` 打印包括字段和限定类型名称在内的实例的完整信息
- `%T` 打印某个类型的完整说明



%t 布尔类型

%T 输出对应位置变量的类型



### [ 格式化输出](https://www.oyohyee.com/post/Note/learn_go/#格式化输出)

输出到控制台使用

```
fmt.Printf()
```

而只需要获得格式化后的字符串使用

```
fmt.sprintf()
```

格式化的替换字符如下:

| 字符     | 含义                                         |
| -------- | -------------------------------------------- |
| `%v`     | 输出结构体的内容                             |
| `%+v`    | 输出结构体的字段名、值                       |
| `%#v`    | 输出结构体的类型、字段名、值                 |
| `%T`     | 输出变量类型                                 |
| `%t`     | 输出布尔值                                   |
| `%d`     | 输出整数                                     |
| `%b`     | 输出二进制整数                               |
| `%c`     | 输出整数对应的ascii字符                      |
| `%x`     | 输出十六进制编码                             |
| `%f`     | 输出浮点数                                   |
| `%e`     | 输出科学计数下的浮点数                       |
| `%E`     | 输出科学计数下的浮点数                       |
| `%s`     | 输出字符串                                   |
| `%p`     | 输出类型                                     |
| `%-a.bf` | 输出左对齐的宽度为`a`且保留`b`位小数的浮点数 |
