# Func

方法的形参,可以只有类型,没有变量名

方法的接收者,也是可以只有类型,没有接收者名

如果没有变量名,那就全部都没有

函数传参都是值拷贝



## Main

main函数不能带参数

main函数不能定义返回值

main函数所在的包必须为main包

main函数中可以使用flag包来获取和解析命令行参数



## Init

init 函数在main函数执行之前执行,

可以声明多个init函数,执行顺序是"先进先出"。队模型





go 支持匿名函数!!!

变量 := func(){...}
